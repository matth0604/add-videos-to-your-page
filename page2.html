Developer Tools:

  • HTML:
Controls structure and is a language of syntax and rules.
  • CSS-Cascading Style Sheet:
CSS changes how things look. Important for larger webpages where multiple pages are involved in order to save time rewriting the HTML code on every page and to keep the same style consistent on every page. CSS also helps prevent errors as computers are "stupid" in a sense that if you miss one closing tag it will cause everything to continue to be styled the way you selected in the element prior. By utilizing CSS a user can edit an entire sites style just by editing a single file.
  • JS-Javascript:
JS is used to add dynamic and interactive elements to websites.
